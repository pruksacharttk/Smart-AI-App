API Keys & Model Fallback

คู่มือฉบับย่อ: สมัครและนำ API key มาใส่

  1. NVIDIA NIM: ไปที่ build.nvidia.com/settings/api-keys แล้ว Sign in ด้วยบัญชี NVIDIA จากนั้นกดสร้าง API key แล้วคัดลอกมาใส่ช่อง NVIDIA NIM ด้านล่าง
  2. OpenRouter: ไปที่ openrouter.ai/settings/keys แล้ว Sign in/Sign up จากนั้นกด Create key แล้วคัดลอกมาใส่ช่อง OpenRouter ด้านล่าง
  3. API key มักแสดงให้คัดลอกเพียงครั้งเดียว ให้เก็บไว้ในที่ปลอดภัย และในแอปนี้ช่อง key จะถูกซ่อนเป็นรหัสผ่านพร้อมปุ่ม Show/Hide
  4. หลังใส่ key ให้เลือก fallback model ลำดับ 1-4 แล้วกด Save Config ค่า config จะถูกเก็บใน browser นี้เท่านั้น ไม่สร้างไฟล์ config และไม่ใช้ database
