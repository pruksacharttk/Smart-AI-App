# Premium Visual Patterns

## Luxury SaaS Hero

- Large editorial headline.
- One high-contrast CTA and one quiet secondary CTA.
- Minimal eyebrow badge.
- Product preview with subtle depth.
- Dark or warm off-white background.
- One controlled signature effect.

## Premium Dashboard

- Metric cards include context, trend, and timeframe.
- Filters live in a toolbar.
- Tables have readable row height and clear row actions.
- Status uses badges plus text, not color alone.
- Charts include captions and fallback states.

## High-End Form

- One clear title and helper paragraph.
- Labels above fields.
- Inline validation.
- Primary action prominent, secondary action quiet.
- Long forms grouped into sections or steps.

## Editorial Landing Page

- Strong grid and controlled asymmetry.
- Oversized section headings.
- Pull quotes or callout cards.
- Generous vertical rhythm.
- Fewer but stronger visual moments.
