# `$visual-ui-enhancement` vs `/visual-ui-enhancement` in Codex

Use:

```text
$visual-ui-enhancement
```

This activates the skill.

Do not assume this activates the skill:

```text
/visual-ui-enhancement
```

Slash commands are a separate command system. `/visual-ui-enhancement` is valid only if you create a custom slash command with that exact name.

Useful Codex command:

```text
/skills
```

Use `/skills` to browse or select installed skills.
