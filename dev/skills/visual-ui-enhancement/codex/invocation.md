# Codex Invocation Guide

## Preferred explicit invocation

```text
$visual-ui-enhancement

Improve the UI in app/dashboard/page.tsx.
Use Tailwind CSS + shadcn/ui.
Focus on premium visual hierarchy, responsive behavior, accessibility, and production readiness.
```

## Implicit invocation

Codex may select the skill automatically when your task matches the skill description, for example:

```text
Make this settings page look more premium and accessible with Tailwind and shadcn/ui.
```

## When to use subagents

Use subagents for bigger tasks, such as redesigning a dashboard, landing page, admin panel, onboarding flow, or large form.

```text
$visual-ui-enhancement

Run this as a Codex subagent workflow with Requirement Analyzer, Visual Direction Agent,
shadcn/Tailwind Builder, UX Reviewer, Accessibility Reviewer, Responsive Reviewer,
and Final Refactor Agent.
```
