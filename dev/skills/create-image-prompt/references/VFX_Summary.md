## Usage Rules

### Do:
- Choose VFX that matches scene mood
- Describe effect naturally in prompt
- Integrate VFX into scene description

### Don't:
- Include internal codes (P/L/W/M/A/T) in final prompt
- Stack multiple VFX unless they complement each other
- Use VFX that conflicts with realism (unless intended)

### VFX + Style Combinations
VFX can be combined with any style:
- Cinematic + Rain = Dramatic movie scene
- Fantasy + Magic Circle = Mystical atmosphere
- Anime + Speed Lines = Dynamic action
- Nature + Mist = Mysterious forest

### Override Note
When VFX is selected, it overrides Realism Default Protocol (RDP).
The prompt shifts from pure photorealism to artistic interpretation.
