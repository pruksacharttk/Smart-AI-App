# Quick Start Guide

Use this skill to generate **one final landscape image prompt**.

## Basic flow
1. Describe the landscape you want.
2. Optionally attach a primary reference image to preserve the house.
3. Choose the output language (`en` or `th`).
4. Keep the default **Mode 5 - Text to Landscape** for most text-only requests.
5. If needed, use **Variation Type** to let the skill internally compare options before returning one final prompt.

## Variation controls
- **Direct Single Prompt**: fastest, returns one prompt directly.
- **Generate Several Candidates, Return the Best One**: the skill drafts several internal candidates, then returns the strongest one.
- **Use a Custom Focus, Then Return One Final Prompt**: choose a clear creative direction and optionally add notes.

## Output format
The result is a **single raw prompt string only**.
No JSON wrapper. No bullet list. No extra explanation.
