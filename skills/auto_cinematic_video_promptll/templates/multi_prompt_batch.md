# Multi Prompt Batch Template

Generate multiple independent Seedance 2-ready cinematic prompts from the same reference identity and style. Each prompt must preserve the same subject, wardrobe, hair, makeup, environment, lighting, and color grade, but must vary the camera placement, composition, pose, gaze, or shot scale.

Each prompt should be self-contained and include:
- Aspect ratio
- Duration
- Camera placement and movement
- Subject action
- Lighting and color grade
- Continuity locks
- Negative prompt

Never output duplicate prompts. Every generated prompt must differ in at least two meaningful axes.
