# Mode: contact_sheet_2x3

Generate one photorealistic 2×3 contact sheet image containing exactly 6 frames. All six frames must preserve the exact same subject identity, wardrobe, styling, hair, makeup, lighting, environment, color grade, and photographic style. The frames must feel like different camera placements within the same scene, not different scenes.

Default six-frame order:
1. High-Fashion Beauty Portrait: close, editorial, intimate.
2. High-Angle Three-Quarter Frame: overhead off-center diagonal.
3. Low-Angle Oblique Full-Body Frame: low ground-level oblique full-body perspective.
4. Side-On Compression Frame: long lens profile or near-profile.
5. Intimate Close Portrait From an Unexpected Height: close face or upper torso, unconventional height.
6. Extreme Detail Frame: abstract but photoreal close-up of real wardrobe, texture, hair, neckline, strap, accessory, shoe, or set detail.

Layout rules:
- 2 rows, 3 columns.
- Same internal frame ratio for all panels.
- Thin clean borders.
- Optional top-left labels: Frame 1 through Frame 6.
- Optional concise captions describing only final camera position and subject state.

Do not invent accessories or details for the extreme detail frame.
